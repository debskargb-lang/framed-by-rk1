FRAMED BY RK — PREMIUM EDITABLE WEBSITE

Open index.html to view.

HOW TO EDIT:
1. Open content.json in any text editor.
2. Change text, prices, product names, FAQs, testimonials, colours and links.
3. Save.
4. Refresh index.html.

HOW TO CHANGE PHOTOS:
1. Add your new image to the assets folder.
2. In content.json, change the image path, for example:
   "image": "assets/my-new-photo.jpg"

HOW TO PUBLISH:
Upload the whole folder to Netlify, Vercel, GitHub Pages, or any web host.

MAIN FILES:
- index.html = page structure
- styles.css = design
- script.js = loads the editable content
- content.json = edit this most of the time
- assets = images/logo
